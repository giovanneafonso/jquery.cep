jQuery CEP Plugin

Um plugin jQuery para pegar endereços de acordo com o CEP informado
  • Demonstração
  • Documentação
  • Opções do Plugin
Digite um CEP acima para pesquisar os dados.

Os dados serão preenchidos automaticamente.

Como Funciona

jQuery CEP Plugin utiliza um webservice de CEP para pegar dados do endereço de acordo com o CEP.

Utilização Básica

O jeito mais simples de utilizar o plugin é criar um formulário de endereço e, em cada campo, colocar os atributos requeridos.

Ver Demo
